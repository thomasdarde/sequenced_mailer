# SequencedMailer

Allow to send emails at a specific time
Allow to send a serie of emails



## Installation & Setup

To install add the following to your Gemfile:

```ruby
  gem 'sequenced_mailer'
```


To setup just run:

```bash
  $ rake sequenced_mailer:setup
```

and follow the guide!
